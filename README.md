# PUP-Hymn Lyrics Player

A web-based application featuring the **PUP Hymn** with synchronized, real-time lyrics highlighting.

## Live Demo
Check out the live project here: 
[https://nicolestaana.github.io/PUP-Hymn/](https://nicolestaana.github.io/PUP-Hymn/)

## Key Features
* **Synchronized Lyrics:** Each line of the hymn highlights automatically in sync with the audio playback.
* **Audio Integration:** Interactive play and pause controls for seamless listening.
* **Clean Interface:** A beginner-friendly, organized layout built using standard web structures.

## Technical Stack
* **HTML5** – Structured using standard `div` blocks for layout consistency.
* **CSS3** – Custom styling for a modern look and feel.
* **JavaScript** – Core logic for audio time-tracking and dynamic text synchronization.

## Project Structure
* `index.html` – The main skeleton of the webpage.
* `style.css` – Design and visual elements of the interface.
* `script.js` – The timing logic for the synchronized lyrics.
* `PUP BG1-4.png` – Background and logo assets.
* `Imno ng PUP.mp3` – The audio source for the hymn.
